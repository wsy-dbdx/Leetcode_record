求硬币组合总数，这种就不是找到一个就可以的，需要找全部的组合，也就是一个硬币从最小的数量开始找，一直到最大数量为止，  
遍历一个硬币开始的所有金额
```JAVA
class Solution {
  public int change(int amount, int[] coins) {
    int[] dp = new int[amount + 1];
    dp[0] = 1;
    for (int coin : coins) {
      for (int x = coin; x < amount + 1; ++x) {
        dp[x] += dp[x - coin];
      }
    }
    return dp[amount];
  }
}
```
